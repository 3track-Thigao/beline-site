# BE.LINE — Guia de Identidade Visual Digital

## Paleta

```css
--carmim:  #8C0502  /* primária — textos fundo branco */
--rubro:   #680A08  /* carmim escuro */
--ebano:   #0d0d0d  /* fundo seções escuras */
--algodao: #ffffff  /* textos fundo escuro */
--ouro:    #9D753E  /* destaques, badges, preços */
```

## Regra de cor

| Fundo | Textos | Destaques |
|-------|--------|-----------|
| Branco | Carmim | Ouro |
| Preto | Algodão | Ouro |

## Tipografia

- **Cormorant Garamond** — títulos, citações, nomes de peças
- **Saira** — logo, labels, eyebrows
- **Poppins** — corpo, botões, preços

## Listras (elemento de identidade)

```css
/* preto/branco */
repeating-linear-gradient(90deg, #0d0d0d 0 8px, #fff 8px 16px)

/* carmim/ouro */
repeating-linear-gradient(90deg, #8C0502 0 10px, #9D753E 10px 20px)
```

## Fotografia

- Coleção ativa: editorial colorida
- Arquivo: preto e branco, brightness 0.7
- Nunca fundo branco de catálogo
